Aula sobre Classes em JavaScript

1. Introdução às Classes em JavaScript

O que são Classes?

Classes em JavaScript são um recurso introduzido no ECMAScript 2015 (ES6) que oferece uma maneira mais clara e concisa de criar objetos e lidar com herança. Embora JavaScript seja uma linguagem baseada em protótipos para herança, as classes oferecem uma sintaxe mais simples e direta para criar objetos e implementar herança.

2. Sintaxe básica de uma Classe

class Pessoa { constructor(nome, idade) { this.nome = nome; this.idade = idade; } seApresentar() { return `Meu nome é ${this.nome} e eu tenho ${this.idade} anos.`; } }

Explicação:

  • class Pessoa declara uma classe chamada Pessoa.
  • constructor(nome, idade) é um método especial para criar e inicializar um objeto criado com uma classe. Ele recebe os parâmetros nome e idade e os associa à nova instância do objeto.
  • seApresentar() é um método da classe que retorna uma string apresentando a pessoa.

3. Criando Instâncias de uma Classe

const pessoa1 = new Pessoa("Alice", 30); console.log(pessoa1.seApresentar()); // Saída: "Meu nome é Alice e eu tenho 30 anos."

4. Herança entre Classes

Herança é um princípio fundamental da programação orientada a objetos que permite a uma classe herdar propriedades e métodos de outra.

class Estudante extends Pessoa { constructor(nome, idade, curso) { super(nome, idade); this.curso = curso; } seApresentar() { return `${super.seApresentar()} Eu estudo ${this.curso}.`; } }

5. Criando uma Instância da Subclasse

const estudante1 = new Estudante("Bob", 20, "Engenharia de Software"); console.log(estudante1.seApresentar()); // Saída: "Meu nome é Bob e eu tenho 20 anos. Eu estudo Engenharia de Software."

6. Boas Práticas ao Usar Classes

  • Encapsulamento: Mantenha dados que não devem ser acessados diretamente privados usando o prefixo # em propriedades e métodos.
  • Reusabilidade: Utilize herança apenas quando faz sentido (relação "é-um").
  • Legibilidade: Nomeie classes, métodos e propriedades de forma clara e intuitiva.

Conclusão:

Classes em JavaScript simplificam a estruturação de códigos mais complexos, tornando o código mais legível e fácil de manter. Com a prática, você começará a ver os benefícios de utilizar essa abordagem em seus projetos de desenvolvimento web.
